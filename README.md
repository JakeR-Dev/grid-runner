# Grid Runner

Play the game at: https://jaker-dev.github.io/grid-runner/

The Goal: collect as many 'signals' as possible without letting the enemies touch you.

Built with:
- React
- Vite
- SCSS

## Compile assets

- Be sure you're running npm 24+
- From project root, run `npm run dev`

## Deployment

- From project root, run `npm run build`
